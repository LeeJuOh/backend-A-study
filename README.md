## 📜 제출 방법

깃 연습을 위데브 코스의스의 룰을 따른다.

```
1. 원본 레포지토리 fork 하기
2. clone 하기
3. 자신의 깃헙 닉네임으로 브랜치를 생성하고 원본 레포지토리로 PR를 보낸다. 
    ex) yhh1056 -> (원본) yhh1056

```
